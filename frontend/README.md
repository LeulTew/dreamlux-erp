# Frontend

This frontend uses Next.js with Bun.

## Local Development

From repository root:

```bash
bun run install:all
bun run dev:frontend
```

Or from this folder:

```bash
bun install
bun run dev
```

## Commands

```bash
bun run dev
bun run build
bun run start
bun run lint
```

## Required Environment Variable

- `NEXT_PUBLIC_API_URL` (for example: `http://localhost:4000` locally)
