# [Epic] Pillar 3: Event Management

**Issue ID**: `EPIC-2`
**Labels**: `type:epic`, `area:events`, `priority:p0`

---

## Description

Implement the complete lifecycle, allocation, and tracking system for Event Management (Pillar 3) as defined in the Dream Lux SRD.

## Key Objectives

- Develop database tables and backend APIs for events, scheduling, vehicle logs, and event expenses.
- Prevent scheduling conflicts for employees and vehicles (blocking double-booking on overlapping dates).
- Build the operational and financial dashboard panels for managing events.
- Implement an automated expense approval queue for accountants/owners.
- Enable detailed profitability calculations (Revenue vs. Approved Expenses).
