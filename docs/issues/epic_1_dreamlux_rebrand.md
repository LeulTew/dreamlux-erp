# [Epic] Dream Lux Rebranding & Styling

**Issue ID**: `EPIC-1`
**Labels**: `type:epic`, `area:rebranding`, `priority:p0`

---

## Description

Complete the transition from the generic "Unified ERP Demo" design to a premium, high-contrast, elegant style tailored specifically for **Dream Lux PLC**.

## Key Objectives

- Establish the core styling theme (elegant gold accents, dark slate/purple dark mode, clean white light mode).
- Standardize geometric controls (minimal border radius, no/minimal shadows, high-contrast borders).
- Eliminate all generic/AI demo placeholders, sci-fi terms, or gaming slop text from the codebase.
- Validate WCAG accessibility contrast ratios on all elements, specifically SVGs, icons, text labels, and active hover states.
